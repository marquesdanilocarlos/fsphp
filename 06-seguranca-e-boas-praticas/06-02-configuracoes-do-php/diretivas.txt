;As diretivas a seguir podem proteger de ataques RFI (Remote File Inclusion), ataque de inclusão remota de arquivos onde o atacante utiliza uma URL para inclusão de um arquivo malicioso, que mostra dados como constantes de senhas ou que reescreve a index por exemplo, gerando um ataque DOS.

;Permite que o servidor abra arquivos externos

;Permite que o código PHP do meu servidor inclua arquivos externos

;Tempo máximo de execução de um script que realizou uma ação de input, como por exemplo um formulário que fez o upload de um arquivo

;Controle de buffer de saída do PHP. Já que cada comando de saída no PHP equivale a uma requisição, é possível guardar todas as saídas num buffer e cuspir na tela somente no final do arquivo, gerando apenas uma requisição.

;Habilita ou desabilita o buffer de saída
;On = Habilita e não seta limite de tamanho para o buffer

;Habilita a execução imediata de um comando de saída no PHP, gerando uma requisição para cada comando
